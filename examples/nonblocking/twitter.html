Twitter Widget SPOF

This page includes a typical twitter share widget, in this case a Tweet Button, properly loading asynchronously from Twitter's documentation.

Compare this page to the SPOF example here.

The Button!

Tweet

The Code

There are two parts to providing a Twitter Button, creating an <a> tag and including the reqired JavaScript. The <a> tag is standard, with a specific CSS class and optional data-size attributes. 

<a class="twitter-share-button" href="https://twitter.com/intent/tweet?text=Hello%20world" data-size="large">Tweet</a>

Follow Twitters documentation and add the following code to the page: 

<script>window.twttr = (function(d, s, id) {
  var js, fjs = d.getElementsByTagName(s)[0],
    t = window.twttr || {};
  if (d.getElementById(id)) return t;
  js = d.createElement(s);
  js.id = id;
  js.src = "https://platform.twitter.com/widgets.js";
  fjs.parentNode.insertBefore(js, fjs);

  t._e = [];
  t.ready = function(f) {
    t._e.push(f);
  };

  return t;
}(document, "script", "twitter-wjs"));</script>
